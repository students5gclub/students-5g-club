Data Analysis Guide

1. Data Cleaning

Real-world data is often messy, with missing values, duplicates, or inconsistencies. Cleaning data ensures accurate and reliable analysis.

Key Concepts:

  • Handling Missing Data: Identify missing values and use techniques like filling with mean/median or dropping rows with excessive missing entries.
  • Dealing with Outliers: Use visualization (e.g., box plots) to identify and handle outliers effectively.
  • Removing Duplicates: Clean duplicate entries to avoid inflated results.

Example (Python):

    
    import pandas as pd
    data = {"Name": ["Alice", "Bob", None], "Age": [25, None, 30]}
    df = pd.DataFrame(data)
    
    # Fill missing values with a default
    df['Age'].fillna(df['Age'].mean(), inplace=True)
    print(df)

2. Data Visualization

Visualization helps uncover patterns and insights in data.

Key Concepts:

  • Types of Visualizations: Bar charts, line graphs, and scatter plots.
  • Popular Tools: Tableau, Power BI, and Matplotlib.

Example (Python):

    
    import matplotlib.pyplot as plt
    
    categories = ['A', 'B', 'C']
    values = [10, 20, 15]
    
    plt.bar(categories, values, color='skyblue')
    plt.title('Bar Chart Example')
    plt.xlabel('Categories')
    plt.ylabel('Values')
    plt.show()

3. Data Insights

Insights help in decision-making and forecasting by extracting meaningful information from data.

Key Concepts:

  • Descriptive Analysis: Summarizes historical data using mean, median, mode, and standard deviation.
  • Predictive Analysis: Forecasts future outcomes using techniques like regression and classification.

Example (Python - Predict Sales):

    
    from sklearn.linear_model import LinearRegression
    import numpy as np
    
    # Example data
    X = np.array([10, 20, 30, 40]).reshape(-1, 1)  # Marketing spend
    y = np.array([100, 200, 300, 400])            # Sales
    
    # Model training
    model = LinearRegression()
    model.fit(X, y)
    
    # Predict
    future_spend = np.array([50]).reshape(-1, 1)
    predicted_sales = model.predict(future_spend)
    print("Predicted Sales:", predicted_sales[0])
