C Learning Guide

1. Getting Started with C Programming

  • C is a powerful, structured programming language used in system programming and performance-critical applications.
  • Set up your environment using a C compiler like GCC or IDEs such as Code::Blocks or Visual Studio.
  • Write and run your first program:
    
    #include <stdio.h>
    
    int main() {
        printf("Hello, World!\n");
        return 0;
    }

2. Data Types and Variables

  • Understand basic data types like int, float, double, and char.
  • Example of declaring variables:
    
    int age = 25;
    float salary = 50000.50;
    char grade = 'A';

3. Operators

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, !=, <, >, <=, >=
  • Logical Operators: &&, ||, !
    
    int a = 10, b = 20;
    if (a < b && b > 15) {
        printf("Condition is true\n");
    }

4. Control Flow (Conditional Statements)

  • If-Else: Use conditions to control program flow.
    
    int age = 18;
    if (age >= 18) {
        printf("Eligible to vote\n");
    } else {
        printf("Not eligible to vote\n");
    }

5. Loops

  • For Loop: Repeats a block of code a fixed number of times.
    
    for (int i = 1; i <= 5; i++) {
        printf("Iteration %d\n", i);
    }
  • While Loop: Runs as long as the condition is true.
    
    int n = 5;
    while (n > 0) {
        printf("%d\n", n);
        n--;
    }

6. Functions

  • Break your program into reusable blocks of code.
    
    #include <stdio.h>
    
    int add(int a, int b) {
        return a + b;
    }
    
    int main() {
        int result = add(10, 20);
        printf("Sum: %d\n", result);
        return 0;
    }

7. Arrays

  • Store multiple values of the same type in an array.
    
    int numbers[5] = {10, 20, 30, 40, 50};
    printf("First element: %d\n", numbers[0]);

8. Strings

  • Strings are arrays of characters ending with a null character \0.
    
    char name[] = "Alice";
    printf("Name: %s\n", name);

9. Pointers

  • Pointers store the address of another variable.
    
    int x = 10;
    int *ptr = &x;
    printf("Value of x: %d\n", *ptr);

10. File Handling

  • Read and write files in C using FILE pointers.
    
    #include <stdio.h>
    
    int main() {
        FILE *file = fopen("data.txt", "w");
        if (file != NULL) {
            fprintf(file, "Hello, File!\n");
            fclose(file);
        }
        return 0;
    }
