C++ Learning Guide

1. Introduction to C++

What is C++?

  • C++ is a general-purpose, high-performance programming language.
  • Used in system software, games, and application development.

Setting Up Your Environment

  • Install an IDE like Code::Blocks, Dev-C++, or Visual Studio.
  • Use online compilers like Replit or GeeksforGeeks IDE for quick practice.

2. Basics of C++

Structure of a C++ Program


    #include <iostream>
    using namespace std;
    
    int main() {
        cout << "Hello, World!";
        return 0;
    }

Comments

  • Single-line: // This is a comment.
  • Multi-line: /* This is a multi-line comment. */

Variables and Data Types

  • Data Types: int, float, char, bool, string.
  • Example:

    int age = 20;
    float height = 5.9;
    char grade = 'A';
    bool isStudent = true;
    string name = "John";

Input and Output


    int age;
    cout << "Enter your age: ";
    cin >> age;
    cout << "Your age is " << age << endl;

3. Control Structures

If-Else Example


    int number;
    cout << "Enter a number: ";
    cin >> number;
    
    if (number > 0) {
        cout << "Positive number";
    } else if (number < 0) {
        cout << "Negative number";
    } else {
        cout << "Zero";
    }

Loops

  • For Loop:
    • 
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
  • While Loop:
    • 
    int i = 1;
    while (i <= 5) {
        cout << i << " ";
        i++;
    }

4. Functions

Example Function


    int add(int a, int b) {
        return a + b;
    }
    
    int main() {
        cout << add(5, 3); // Output: 8
        return 0;
    }

5. Object-Oriented Programming (OOP)

Classes and Objects


    class Car {
    public:
        string brand;
        int speed;
    
        void display() {
            cout << "Brand: " << brand << ", Speed: " << speed;
        }
    };
    
    int main() {
        Car myCar;
        myCar.brand = "Toyota";
        myCar.speed = 120;
        myCar.display();
        return 0;
    }

  • Reverse a string using loops.
  • Write a class for a bank account system.
